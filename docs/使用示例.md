# 使用示例

## 1. 自动登录流程（已实现）

应用启动后会自动检查登录状态：

```java
// MainActivity.onCreate()
SharedPreferences prefs = getSharedPreferences("LoginPrefs", MODE_PRIVATE);
boolean isLoggedIn = prefs.getBoolean("isLoggedIn", false);

if (isLoggedIn) {
    // 已登录，加载主页面
    String jsessionId = prefs.getString("JSESSIONID", "");
    loadMainPage();
} else {
    // 未登录，启动登录流程
    startLogin();
}
```

## 2. 手动启动登录流程

如果需要在其他地方手动启动登录：

```java
// 在任何 Activity 中
private final ActivityResultLauncher<Intent> loginLauncher = registerForActivityResult(
    new ActivityResultContracts.StartActivityForResult(),
    result -> {
        if (result.getResultCode() == RESULT_OK && result.getData() != null) {
            String jsessionId = result.getData().getStringExtra(LoginActivity.EXTRA_JSESSIONID);
            Log.d(TAG, "Login successful, JSESSIONID: " + jsessionId);
            // 处理登录成功
        }
    }
);

// 启动登录
private void startLogin() {
    Intent intent = new Intent(this, LoginActivity.class);
    loginLauncher.launch(intent);
}
```

## 3. 在 JavaScript 中使用

```javascript
// 检查登录状态
if (Bridge.isLogin()) {
    console.log('用户已登录');
    
    // 获取 JSESSIONID
    var jsessionId = Bridge.getJSessionId();
    console.log('JSESSIONID: ' + jsessionId);
    
    // 使用 JSESSIONID 进行 API 请求
    fetch('https://api.example.com/data', {
        headers: {
            'Cookie': 'JSESSIONID=' + jsessionId
        }
    });
} else {
    console.log('用户未登录');
}
```

## 4. 提取 JSESSIONID 工具方法

```java
// 从 cookie 字符串中提取 JSESSIONID
String cookies = "JSESSIONID=ABC123DEF456; Path=/; HttpOnly";
String jsessionId = login.extractJSessionId(cookies);
System.out.println("JSESSIONID: " + jsessionId); // 输出: ABC123DEF456
```

## 5. 登出功能

```java
// 清除登录状态
SharedPreferences prefs = getSharedPreferences("LoginPrefs", MODE_PRIVATE);
SharedPreferences.Editor editor = prefs.edit();
editor.putBoolean("isLoggedIn", false);
editor.remove("jsessionId");
editor.remove("cookies");
editor.apply();

// 重启应用或跳转到登录页面
Intent intent = new Intent(this, MainActivity.class);
intent.setFlags(Intent.FLAG_ACTIVITY_NEW_TASK | Intent.FLAG_ACTIVITY_CLEAR_TASK);
startActivity(intent);
```

## 7. 错误处理

```java
// 在 LoginActivity 中处理登录失败
@Override
public void onBackPressed() {
    if (webView.canGoBack()) {
        webView.goBack();
    } else {
        // 用户取消登录
        Log.d(TAG, "User cancelled login");
        setResult(RESULT_CANCELED);
        super.onBackPressed();
    }
}

// 在 MainActivity 中处理取消
loginLauncher = registerForActivityResult(
    new ActivityResultContracts.StartActivityForResult(),
    result -> {
        if (result.getResultCode() == RESULT_OK) {
            // 登录成功
        } else {
            // 登录取消或失败
            Toast.makeText(this, "登录已取消", Toast.LENGTH_SHORT).show();
            // 可以选择关闭应用或重试
        }
    }
);
```

## 9. 常见问题

### Q: 如何重新登录？
A: 点击"登出"按钮，然后重启应用。或者使用 `adb shell pm clear` 清除应用数据。

### Q: JSESSIONID 有效期是多久？
A: 由服务器决定，通常是会话级别的 Cookie，浏览器关闭后失效。

### Q: 如何在后台任务中使用 JSESSIONID？
A: 从 SharedPreferences 中读取：
```java
SharedPreferences prefs = context.getSharedPreferences("LoginPrefs", Context.MODE_PRIVATE);
String jsessionId = prefs.getString("jsessionId", "");
```

### Q: 为什么有时候提取不到 JSESSIONID？
A: 可能原因：
1. Cookie 格式不标准
2. 服务器没有设置 JSESSIONID
3. URL 不匹配（检查日志）
4. 网络请求失败

### Q: 如何支持其他 Cookie？
A: 修改 `login.extractJSessionId()` 方法，添加其他 Cookie 的提取逻辑。
