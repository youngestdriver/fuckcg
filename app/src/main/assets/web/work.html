About

关于本工具

本工具用于自主生成 json 数据，并使用官方加密算法签名后提交，以达到假装跑步的目的

实现介绍

1. 将创高体育2.9.6脱壳后逆向，找到生成签名的算法，发现核心签名算法在 so 库中；

2. 使用 ida pro 逆向并修改 so 库中的签名验证逻辑，使得在我们自行开发的应用中也能生成签名；

3. 另外，使用抓包软件发现接口并伪造 http 请求提交。
